## Visual Studio Code 设置运行 Python 文件的快捷键

1. 按下 **`Command + K + S`** 打开键盘快捷键设置
2. 输入 **`Python: Run Python File in Terminal`** 并选择它
3. 在 **`Keybinding`** 设置 ***`Shift + Enter`*** 作为运行代码的快捷键
